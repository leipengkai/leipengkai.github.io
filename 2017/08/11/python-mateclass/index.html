python元类

2017-08-11
python3

元类就是类的类,是用来创建这些类(对象)的

类就是一组用来描述如何生成一个对象的代码段(数据结构和操作这些数据结构的方法组成)
但python中 类也是对象

类本身以及由类创建出来的实例统统都称之为对象,python中一切皆对象,unix一切皆文件

Python中所有的东⻄,指所有的东⻄——都是对象.这包括整数、字符串、函数以及类

不推荐用元类创建类,还是用class创建类
